# Core.js
Alternative standart library for javascript. Includes polyfills for ECMAScript 5, ECMAScript 6, setImmediate, static array methods, extended object api and other sugar.
### Install
```
npm i core-js
```
### Usage
Browser:
```html
<script src='core.min.js'></script>
```
Node.js:
```javascript
require('core-js');
```
